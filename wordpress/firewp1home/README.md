firewp1home is a WordPress-native home screen plugin for AW Roadside.

Shortcode:

```text
[firewp1home]
```

Purpose:
- Render a clean site-first home screen inside WordPress
- Load only the home screen UI
- Proxy API calls to the Render backend
- Keep customer/provider/admin screens separate from the home entry screen

After activation:
1. Set the Render Base URL in the plugin settings page.
2. Add the shortcode to the target WordPress page.
